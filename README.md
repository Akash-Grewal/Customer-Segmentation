📊 Customer Segmentation Using RFM Analysis.

📌 Overview This project performs customer segmentation using RFM (Recency, Frequency, Monetary) Analysis in Microsoft Excel. It helps businesses identify and target different customer groups based on their purchase behavior.

🎯 Objective

Segment customers into meaningful groups
Identify high-value, at-risk, and lost customers
Enable targeted marketing strategies
🛠️ Tools Used

Microsoft Excel
RFM Analysis Framework
Pivot Tables & Charts
